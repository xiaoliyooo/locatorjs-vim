üîß Installation Guide

  1. Enable Kitty Remote Control

    Add this to your ~/.config/kitty/kitty.conf:

    allow_remote_control yes listen_on unix:/tmp/locator_vim_kitty--
  2. Install the Native Host

    Run the install script from the extension directory:

    cd /path/to/locatorjs-vim chmod +x install_host.sh ./install_host.sh
  3. Reload Chrome Extensions

    Go to chrome://extensions, enable Developer Mode, and reload the extension.

  4. Restart Kitty

    Restart your Kitty terminal to apply the remote control settings.
